Pomodoro App


### アプリ概要

生産性向上アプリ。自分だけのtodoを追加して、決められた時間以内にそのtodoを完了させるために、タイマー機能がついている。
1980年代に大学教授のFrancesco Cirillo氏が始めたPomodoro Techniqueが基になっている。
この技術の基本的なルールは、課題を細分化し、一つ一つの子課題を25分以内に完了させ、5分休憩、また25分をスタートさせるというものであった。Cirillo氏が25分を計るタイマーがトマト＝ポモドーロ（イタリア語）を模したキッチンタイマーであったことからこの名がついた。

このアプリでは、5, 15, 25, 50分のいずれかの長さのポモドーロセッションを自分で選び課題に向かうことができるようになっている。
